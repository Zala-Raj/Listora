# Listora 🎌

Listora is a modern offline-first anime tracking application designed for anime fans who want a simple and organized way to manage their watchlists without accounts, cloud sync, or unnecessary complexity.

Track anime progress, ratings, favorites, and viewing statistics in a clean and responsive interface built for both dark and light themes.

## ✨ Features

- Offline-first architecture
- No login or signup required
- Manual anime tracking
- Episode progress tracking
- Watch status management
- Favorites system
- Rating support (1–10)
- Anime statistics dashboard
- Genre categorization
- JSON Import / Export backups
- Dark & Light theme support
- Smooth and modern UI

## 📱 Screens

- Home Dashboard
- Anime Library
- Statistics & Analytics
- Settings

## 📦 Storage

- Local database storage
- JSON backup system
- Import / Export support for easy migration

## 🎯 Purpose

Listora was created to make anime tracking simple, private, fast, and fully offline while keeping the experience clean and distraction-free.

## 🚫 Not Included

- No accounts
- No cloud sync
- No anime posters/images
- No external APIs
- No ads

## 👨‍💻 About This Project

Listora is my first app and also my first project developed almost entirely using Google AI Studio.

I wanted hands-on practice building a complete application and learning how app development works from idea to finished product. Since I personally found it difficult and boring to organize my anime watchlist manually, I decided to solve that problem by creating this app.

This project started as a practice project, but it helped me learn about UI design, local storage, offline-first apps, data management, and building complete user experiences.

I know there is still a lot to improve, and I plan to build more meaningful and useful projects in the future.

Thank you for checking out Listora and supporting my learning journey ❤️## 🛠 Built For

## 📥 How to Install Listora

1. Open this repository and go to the Releases section.
2. Download the latest Listora APK file from the newest release.
3. Open the downloaded APK file on your Android device.
4. If prompted, enable Install from Unknown Sources or allow installation from your browser/file manager.
5. Tap Install and wait for installation to complete.
6. Open Listora and start organizing your anime watchlist.

Important Notes:

- Currently supported only on Android devices
- Make sure you download APK files only from official releases
- If updating from an older version, exporting your data before updating is recommended

If you face installation issues, bugs, or have suggestions, feel free to contact me.
